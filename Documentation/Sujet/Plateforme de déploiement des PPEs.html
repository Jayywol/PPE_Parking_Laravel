1  Vos obligations

Vous avez l’avez l’obligation pour les CCFs :

  • De déployer vos bases de données sur le serveur.
  • De déployer toutes vos applications web sur le serveur.

Vous devrez nous fournir une fiche de présentation par situation professionnelle. Vous en trouverez un exemple ici.

2  Identifiants de connexion

Vos identifiants de connexion sont initialement les mêmes sur tous les serveurs.

2.1  Login

Votre login est formé de la première lettre de votre prénom suivi de votre nom de famille, sans espace ni caractère spécial. Par exemple :

  • amesle pour Alexandre Meslé
  • aescriva pour Madame Escriva

2.2  Password

Votre password par défaut est azerty, il est très fortement conseillé de le modifier à la première connexion.

3  Serveur Web

Vous trouverez le serveur web à l’adresse 192.168.60.200. Pour déployer une application web, il vous suffit de placer les fichiers sources dans le répertoire ~/public_html.

3.1  Espace Web

Votre espace web est visible à l’URL 192.168.60.200/~login.

3.2  SSH

Connectez-vous une première fois en SSH pour modifier votre mot de passe.

  • Sous linux, utilisez la commande ssh login@192.168.60.200
  • Sous windows, utilisez le logiciel putty.

Le mot de passe se modifie avec la commande passwd.

3.3  Filezilla

Vous pouvez aussi vous connecter à l’aide d’un client FTP en utilisant l’URL 192.168.60.200.

4  Serveur de base de données

Le serveur de bases de données se trouve à l’adresse 192.168.60.201.

4.1  phpmyadmin

Phpmyadmin se trouve à l’adresse http://192.168.60.201/phpmyadmin. Vos identifiants de connexion sont par défaut les mêmes.

N’oubliez pas de modifier le mot de passe !

  • —→Page d’accueil
  • —→Paramètres généraux
  • —→Modifier le mot de passe

4.2  Ligne de commande

Depuis votre ordinateur personnel ou le serveur web, vous pouvez vous connecter à mysql avec la commande mysql -u login -h 192.168.60.201 -p

4.3  Depuis une bibliothèque

Pour vous connecter depuis PDO ou JDBC :

  • host : 192.168.60.201
  • db : votre login
  • Mêmes login et password que pour accéder à phpmyadmin.
